## 接入有多简单？看代码

Java开发者最关注的永远是代码怎么写。我直接给你看实战例子。

最基础的做法，是直接用 `OkHttp` 或 `RestTemplate` 调用千聚的API。假设你原来调百度文心的代码写了好几屏，现在你只需要知道这一个端点：

**示例：用 OkHttp 调用[千聚api聚合平台](https://www.qianjuai.com/)**

java
import okhttp3.*;
import org.json.JSONObject;
import java.io.IOException;

public class QianJuApiDemo {
    // [千聚api聚合平台](https://www.qianjuai.com/)提供的密钥
    private static final String API_KEY = "your_qianju_api_key_here";
    // [千聚api聚合平台](https://www.qianjuai.com/)的统一端点
    private static final String API_URL = "https://www.qianjuai.com/v1/chat/completions";

    public static void main(String[] args) throws IOException {
        OkHttpClient client = new OkHttpClient();

        // 构建与OpenAI完全一致的请求体
        JSONObject requestBody = new JSONObject();
        requestBody.put("model", "gpt-3.5-turbo"); // 想换DeepSeek? 改这里: deepseek-chat
        requestBody.put("messages", new JSONArray()
                .put(new JSONObject()
                        .put("role", "system")
                        .put("content", "You are a helpful assistant."))
                .put(new JSONObject()
                        .put("role", "user")
                        .put("content", "用Java写一个单例模式的例子"))
        );

        Request request = new Request.Builder()
                .url(API_URL)
                .addHeader("Authorization", "Bearer " + API_KEY)
                .addHeader("Content-Type", "application/json")
                .post(RequestBody.create(
                        requestBody.toString(), 
                        MediaType.parse("application/json; charset=utf-8")
                ))
                .build();

        try (Response response = client.newCall(request).execute()) {
            String responseBody = response.body().string();
            System.out.println(responseBody);
        }
    }
}


看到没？就这段代码。它跟调OpenAI的代码唯一区别就是 `base_url` 和 `API_KEY`。你要想换成百度ERNIE、DeepSeek、或者Claude，只需要改请求体里的 `model` 参数值就行了。剩下的鉴权、网络、封装全都复用，一个字都不用改。

对于用 Spring Boot 的项目，更简单，直接往 `RestTemplate` 或 `WebClient` 里塞这个地址就行。再配合 `application.yml` 里的配置，一个密钥走天下。
