## 常见报错及解决办法

如果你运行时报错了，别慌。下面是常见错误及其解决办法：

| 报错信息 | 原因 | 解决方案 |
|------------|-------|---------|
| `AuthenticationError: No API key provided` | API Key未填写或为空 | 检查代码中 `api_key` 是否正确复制 |
| `NotFoundError: 404 model not found` | 模型名写错了 | 确认模型ID为 `claude-3-7-sonnet-20250219` |
| `APIConnectionError: Connection error` | 网络不通或代理问题 | 检查是否能访问 `www.qianjuai.com`，确保不用代理 |
| `RateLimitError: Rate limit exceeded` | 调用频率过高 | 加入 `time.sleep()` 控制请求间隔 |
| `AuthenticationError: Authentication failed` | API Key无效或过期 | 重新去后台创建新的API Key |

**调试清单（按顺序排查）：**

1. ✅ API Key是否正确复制？（前后不能有多余空格）
2. ✅ `base_url` 是否写对了？（必须是 `https://www.qianjuai.com/v1`）
3. ✅ 是否有网络代理干扰？（代码里不应该设置任何 `http_proxy` 或 `https_proxy`）
4. ✅ 模型名是否写对了？（Claude 3.7 Sonnet的模型ID不能写错）
5. ✅ `openai` 库是否是最新版？（建议 `>=1.0.0`）
