## 按量付费 vs 包月：核心差异在哪

先说清楚两个概念。

**按量付费**就是用了多少算多少，用 1 个 token 花 1 分钱。灵活，用不上就不花钱，但单价通常更高，适合用量波动大、项目不确定的情况。

**包月套餐**则是一次性付固定费用，在有效期内享受一个相对低的 token 单价。稳定、可控，但存在“用不完就浪费”的风险，适合用量稳定且可预测的开发者。

举个例子，OpenAI 官方对于 GPT-4 的 API 定价，按量是每 1K 输入 token 收费 0.03 美元，而包月套餐（比如 $20/月）能拿到更低的使用费率。但包月模式不总是对所有人都划算——如果你一个月只跑几千 token，那显然按量付费亏得少。
