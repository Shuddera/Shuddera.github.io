### 第三步：配置Java代码（核心代码在此）

这是最核心的Java示例代码，请完整复制到你的项目中。

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import java.time.Duration;
import java.util.Arrays;

public class ChatGptDemo {
    public static void main(String[] args) {

        // 1. 设置你的千聚API Key
        String yourApiKey = "此处填写你从千聚获取的sk-xxx密钥";

        // 2. 【关键一步】重定向到千聚的网关
        //    这就是实现“无需代理”的秘诀
        String baseUrl = "https://www.qianjuai.com/v1";

        // 3. 构建OpenAiService实例，设置超时时间
        OpenAiService service = new OpenAiService(
                baseUrl,
                yourApiKey,
                Duration.ofSeconds(60)
        );

        // 4. 构建请求消息
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("gpt-4o-mini") // 推荐使用gpt-4o-mini，性价比高，入门首选
                .messages(Arrays.asList(
                        new ChatMessage("system", "你是一个资深的Java技术顾问"),
                        new ChatMessage("user", "请用一句简单的Java代码展示如何连接数据库")
                ))
                .temperature(0.7) // 创造性参数，0.0~2.0
                .maxTokens(150)    // 限制输出长度
                .build();

        // 5. 发起调用并打印结果
        service.createChatCompletion(request)
                .getChoices()
                .forEach(choice -> {
                    System.out.println("ChatGPT回复：");
                    System.out.println(choice.getMessage().getContent());
                });
    }
}


**代码解析与避坑指南：**

-   **`baseUrl = "https://www.qianjuai.com/v1"`**：这一行是灵魂。你不需要改别的东西，只需要把原本指向`api.openai.com`的地址，换成这个**千聚**提供的地址。这才是整个零门槛接入的核心动作。
-   **`model = "gpt-4o-mini"`**：这是**高性价比**的选择。千万不要在第一次测试时就去选`gpt-4o`，跑通流程后，你再慢慢尝试其他模型。
-   **异常处理缺失**：以上是精简版。在生产环境代码中，务必添加`try-catch`块来捕获`OpenAiHttpException`，里面包含详细的错误信息（如速率限制、密钥错误等）。

**千万不要**在代码里硬编码`maxTokens`的值为一个极大数（比如4096）。因为这是**输出Token数的上限**，设置太大，即使你的回复很短，也会消耗满额，导致费用急剧增加。通常，设置`150`到`500`之间是合理的。
