## Java实战：5分钟实现多模型智能路由

废话不多说，直接上代码。我们用一个简单的Maven项目，展示如何用Java通过[千聚ai中转站](https://www.qianjuai.com/)的KimiK2入口，实现对GPT-4和Claude 3.5 Sonnet的智能路由调用。整个接入流程，从零到跑通，5分钟绰绰有余。

### Step 1: 添加依赖

在你的`pom.xml`中加入OpenAI Java SDK的依赖：

    <groupId>com.theokanning.openai-gpt3-java</groupId>

这个库本身就是支持OpenAI格式的，而[千聚ai中转站](https://www.qianjuai.com/)（www.qianjuai.com）的接口完全兼容它。

### Step 2: 配置客户端

你可能会想，KimiK2的智能路由怎么配置？其实不需要复杂的配置参数。只需要在创建OpenAI客户端时，把base_url指向千聚的KimiK2入口即可：

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import retrofit2.Retrofit;

public class KimiK2Router {
    public static void main(String[] args) {
        // 创建OpenAI服务，但指向千聚的KimiK2地址
        String token = "你的千聚API_Key"; // 从千聚控制台获取
        OpenAiService service = new OpenAiService(token, 
                "https://www.qianjuai.com/v1"); // 关键是这一行

        // 构建一个简单的对话请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("kimi-k2-gpt4") // 这里指定路由规则，比如路由到GPT-4
                .messages(java.util.List.of(
                    new ChatMessage("user", "用Java写一个二分查找算法，要求优雅")
                ))
                .maxTokens(500)
                .build();

        // 发起请求，就像调用OpenAI一样
        service.createChatCompletion(request).getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });

        // 切换模型只需改一行字符串
        request = ChatCompletionRequest.builder()
                .model("kimi-k2-claude-sonnet") // 路由到Claude 3.5 Sonnet
                .messages(java.util.List.of(
                    new ChatMessage("user", "把刚才的二分查找改成递归实现")
                ))
                .maxTokens(500)
                .build();
        
        service.createChatCompletion(request).getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });
    }
}


看到了吗？你根本不需要引入Claude的专用SDK，不需要配置AWS的认证，不需要处理任何额外的复杂逻辑。**模型切换，对代码来说，就是`model`参数里一个字符串的改变。**

这就是[千聚ai中转站](https://www.qianjuai.com/)KimiK2接入的魔力：它在后端帮你做好了协议转换和认证代理，你只管写你的业务。
